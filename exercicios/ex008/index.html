Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag "B" Tag(A tag "B" é "não semântica")

Nessa trase temos um termo em destaque usando a tag "strong" (A tag "strong" é semântica

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag "i" (A tag "I" é "não semântica")

Nessa frase temos um temo em itálico usando a tag "EM" (A tag "EM" é semântica)

Texto marcado

Nesta frase temos um texto marcado usando a tag "mark". Para mudar a cor da marcação, usar o parâmetro "style" (dentro da tag mark ou, de preferência, criar um "style" dentro da head do código, digitar o parâmetro que quer mudar, entre aspas Digitar "background-color" e escolher / digitar a cor)

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesta frase

Texto deletado

Podemos marcar um texto como exluido para indicar que ele deve ser ignorado usando a tag "del"

Texto inserido

Podemos marcar um texto como inserido para mostrar que ele foi inserido usando a tag "ins". Essa tag sublinha o texto como a tag "u" (a tag "u" está caindo em desuso)
